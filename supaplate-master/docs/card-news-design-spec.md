# 📋 HTML/CSS 카드뉴스 슬라이드 디자인 지시서

## 프로젝트 개요
썬데이허그 아기 수면 환경 분석 서비스의 **인스타그램 카드뉴스 슬라이드 6장**을 HTML/CSS로 제작해주세요.

## 슬라이드 크기
- **가로: 1080px**
- **세로: 1350px**
- 인스타그램 피드 최적화 비율 (4:5)

## 디자인 컨셉
- **배경색**: 베이지/크림색 (#F5F0E8)
- **포인트색**: 하늘색/민트 (#5BB5C5)
- **텍스트색**: 진회색 (#1a1a1a)
- **폰트**: Cafe24 PROUP (타이틀), Pretendard (본문)
- **캐릭터**: 양 캐릭터 이미지 사용 (sheep.png 제공됨)
- **브랜드**: SUNDAY HUG 로고 (logo.png 제공됨)

---

## 슬라이드 1: 썸네일

**헤더**: 분홍색/핑크색 바 (#E91E63 계열)에 "OO이의 수면환경 기록 일지" 타이틀

**레이아웃**:
- 왼쪽: 아기 사진 (기울어진 프레임, 약 -5도 회전)
- 날짜 표시 배지 (초록색 배경): "12월 10일 (수)"
- 오른쪽 사이드: 컬러 아이콘 3개 (구름/하늘색, 하트/노란색, 돋보기/파란색)
- 하단 왼쪽: 양 캐릭터 + 말풍선 "오늘은 꿀잠자기 목표!"
- 하단 중앙: "SUNDAY HUG" 로고

**동적 데이터**:
```
{{babyName}} - 아기 이름
{{date}} - 날짜
{{babyImage}} - 아기 사진 URL
```

## 슬라이드 2: 엄마의 현실일기

**헤더**: 하늘색 바에 "Date. 2025.12.10 (수) weather. 내 마음의 비."

**레이아웃**:
- 왼쪽: 아기 사진 (직사각형)
- 오른쪽: 줄노트 스타일 일기
  - 회색 가로줄이 있는 노트 형태
  - 각 줄에 일기 텍스트

**일기 예시**:
```
오늘은 뭔가 이상했다

평소에 옆잠베개 해주면

잘 자던 우리 OO이..

하루종일 칭얼거려서

안아주고 그래도 봤지만

방법을 모르겠다..ㅠㅠㅠ

왜 안자는거야... ?? 하루종일 눈 말똥말똥..

그래도 귀엽긴 해..
```

**동적 데이터**:
```
{{date}} - 날짜
{{babyName}} - 아기 이름
{{babyImage}} - 아기 사진 URL
{{momsDiary}} - AI가 생성한 일기 텍스트
```

## 슬라이드 3: 이미지 + 핀 + 점수

**헤더**: 하늘색 바 + 신호등 아이콘(빨/노/초) + "우리 아이의 수면 환경은?"

**레이아웃**:
- 중앙: 아기 전체 사진 (베이지 테두리)
- 사진 위에 번호 핀 오버레이 (1, 2, 3, 4... 분홍색 원)
  - 각 핀은 위험 요소 위치에 배치
- 하단 왼쪽: 양 캐릭터
- 양 옆에 말풍선: "OO이 수면환경 점수는 54점이에요.."

**동적 데이터**:
```
{{babyName}} - 아기 이름
{{babyImage}} - 아기 사진 URL
{{safetyScore}} - 안전 점수 (0-100)
{{feedbackItems}} - 핀 위치 배열 [{id: 1, x: 50, y: 30}, ...]
```

## 슬라이드 4: Bad 피드백 + 추천 제품

**헤더**: 하늘색 바 + 신호등 아이콘 + "환경 주의가 필요해요"

**레이아웃 상단 (bad 섹션)**:
- "bad" 타이틀 (손글씨 스타일)
- 3열 그리드: 아기 사진 크롭 3개
- 각 사진 아래:
  - 분홍색 원 번호 (1, 2, 3)
  - 제목 (예: "누빔패드")
  - 설명 텍스트

**레이아웃 하단 (추천 제품 섹션)**:
- "추천 제품" 타이틀 (손글씨 스타일)
- 3열 그리드: 제품 이미지 3개
  - 제품명 배지 (상단)
  - 제품 이미지
  - 제품명 + 설명

**추천 제품 예시**:
- 백색 소음기: "침대위에는 조용한 소음기만"
- ABC 접이식 아기침대: "단단하고 평평한 침대추천"
- 슬리핑백: "이불 대신 발끝까지 따뜻하게"

**동적 데이터**:
```
{{feedbackItems}} - 피드백 배열 [{id, title, feedback, riskLevel}, ...]
{{recommendedProducts}} - 추천 제품 배열 [{name, imageUrl, description}, ...]
{{babyImage}} - 크롭용 아기 사진
```

## 슬라이드 5: Good 피드백 + 총평

**헤더**: 하늘색 바 + 신호등 아이콘 + "아주 잘하고 있어요!"

**레이아웃 상단 (good 섹션)**:
- "good" 타이틀 (손글씨 스타일)
- 3열 그리드: 아기 사진 크롭 3개
- 각 사진 아래:
  - 분홍색 원 번호
  - 제목
  - 설명 텍스트

**레이아웃 하단 (총평)**:
- 양 캐릭터 (왼쪽)
- 말풍선 (오른쪽): 총평 텍스트

**총평 예시**:
```
전반적으로 아기 수면환경은 양호해요
11개월 된 아기는 활동량이 많아질 수
있어 침대 주변 정리하는 등 몇가지 개선만
하면 더욱 안전한 환경이 될 수 있어요!
```

**동적 데이터**:
```
{{feedbackItems}} - Good 피드백 배열
{{summary}} - AI가 생성한 총평
{{babyImage}} - 크롭용 아기 사진
```

## 슬라이드 6: CTA (마지막)

**레이아웃**:
- 상단: "우리 아기 수면 AI가 다정하게 알려드려요" 타이틀
- 중앙: 하늘색 곡선 장식 (유기적인 blob 형태)
- 곡선 안에: 기울어진 아기 사진 + 양 캐릭터
- CTA 버튼: 검은색 바에 "썬데이허그 AI 수면 환경 분석 받으러 가기"
- 하단: 설명 텍스트 + SUNDAY HUG 로고

**설명 텍스트**:
```
한 장의 사진만 올리면 바로 분석!
전문가가 짚어주는 맞춤 가이드 제공
```

**동적 데이터**:
```
{{babyImage}} - 아기 사진 URL
```
